# fastforge website

The official Fastforge website built with [TanStack Start](https://tanstack.com/start).

## Tech Stack

- **Framework:** [TanStack Start](https://tanstack.com/start) (React)
- **Routing:** [TanStack Router](https://tanstack.com/router)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Content:** MDX/Markdown blog posts

## Development

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev

# Build for production
pnpm build

# Preview production build
pnpm preview
```

## Project Structure

```
src/
  routes/        # File-based routes
  components/    # Shared components
  lib/          # Utility functions
content/
  blog/         # Blog posts (MDX/Markdown)
```

## Blog

Blog posts are located in `content/blog/`. Add a new `.md` or `.mdx` file to create a new post.
