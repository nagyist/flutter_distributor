# S3-compatible Storage

English | [简体中文](../../zh-Hans/publishers/s3.md)

Fastforge uploads files through S3-compatible APIs. The current publishers share one S3 Signature V4 implementation covering S3, MinIO, Qiniu, Alibaba Cloud OSS, and Tencent Cloud COS.

## General Arguments

| Argument           | Description                         |
| ------------------ | ----------------------------------- |
| `endpoint`         | S3-compatible API endpoint          |
| `region`           | Region                              |
| `access-key`       | Access key                          |
| `secret-key`       | Secret key                          |
| `bucket`           | Bucket name                         |
| `key-prefix`       | Optional object key prefix          |
| `public-base-url`  | Optional public download URL prefix |
| `force-path-style` | Whether to use path-style URLs      |

## S3 / MinIO

Required environment variables:

```bash
export S3_ENDPOINT=https://s3.example.com
export S3_REGION=us-east-1
export S3_ACCESS_KEY=access-key
export S3_SECRET_KEY=secret-key
export S3_BUCKET=downloads
```

```bash
fastforge publish --path dist/app.zip --target s3 \
  --publish-arg key-prefix=releases/1.0.0
```

`minio` is an alias for the `s3` target. Standard AWS variables `AWS_REGION`, `AWS_ACCESS_KEY_ID`, `AWS_SECRET_ACCESS_KEY`, and `AWS_SESSION_TOKEN` are also supported.

## Qiniu

```bash
export QINIU_ACCESS_KEY=access-key
export QINIU_SECRET_KEY=secret-key
export QINIU_BUCKET=downloads
export QINIU_REGION=cn-east-1
export QINIU_PUBLIC_BASE_URL=https://download.example.com

fastforge publish --path dist/app.zip --target qiniu
```

The region defaults to `cn-east-1`, and the endpoint is generated from the region by default.

## Alibaba Cloud OSS

```bash
export OSS_ACCESS_KEY=access-key
export OSS_SECRET_KEY=secret-key
export OSS_BUCKET=downloads
export OSS_REGION=cn-hangzhou

fastforge publish --path dist/app.zip --target oss
```

The region is required. The endpoint defaults to `oss-<region>.aliyuncs.com`.

## Tencent Cloud COS

```bash
export COS_ACCESS_KEY=access-key
export COS_SECRET_KEY=secret-key
export COS_BUCKET=downloads-1250000000
export COS_REGION=ap-guangzhou

fastforge publish --path dist/app.zip --target cos
```

The region is required. The endpoint defaults to `cos.<region>.myqcloud.com`.

Provide access keys and secret keys through environment variables only whenever possible.
