# Sample Assets

These files are generated for `dmg_maker` examples and tests.

- `SampleApp.app` is a minimal macOS app bundle used as a fixture.
- `SampleBackground.png` and `SampleBackground@2x.png` are repository-local demo backgrounds.
- `SampleIcon.icns` is generated from the repository-local icon source.
- `Readme.txt` is an extra file used by the sample DMG layouts.
